# OPC Example

Example OPC client and server

## create the virtual environment

```
$ virtualenv venv
$ source venv/bin/activate`
```

## Install the required dependencies

`$ pip install -r requirements.txt`


## Run Server

```
$ cd examples
$ python server-example.py
```

## Run Client

```
$ python client-example.py
```
